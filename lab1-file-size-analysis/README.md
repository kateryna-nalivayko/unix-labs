# Лабораторна робота №1: Частотний аналіз розподілу розмірів файлів

## Опис завдання

Мета роботи — проаналізувати частотний розподіл розмірів файлів у файловій системі UNIX. Потрібно:
1. Зібрати дані про розміри всіх файлів у файловій системі
2. Побудувати візуалізації розподілу (гістограма, CDF, boxplot)
3. Сформулювати висновки про характер розподілу

## Підготовка даних

### Метод збору

Рекурсивний обхід файлової системи починаючи з кореня `/` за допомогою `os.scandir()`. Для кожного файлу зчитується його розмір через `stat()`. Результати стрімінгом записуються у файл `data/result.txt` — один розмір на рядок.

Обробка помилок:
- Символічні посилання пропускаються
- Помилки доступу (`PermissionError`) ігноруються
- Інші помилки ОС (`OSError`) ігноруються

Еквівалент одним рядком у shell:
```bash
find / -type f -exec stat -f '%z' {} + 2>/dev/null > data/result.txt
```

### Формат даних

```
data/result.txt
```

Кожний рядок містить розмір одного файлу в байтах:
```
4096
128
1048576
...
```

## Архітектура рішення

```
collector (os.scandir) → data/result.txt → analyzer (numpy) → plots/ + console output
```

Модулі:
- `core/collector.py` — збір розмірів файлів з файлової системи
- `core/analyzer.py` — завантаження даних, обчислення статистики, висновки
- `core/visualizer.py` — побудова графіків (гістограма, CDF, boxplot)

## Аналіз та візуалізація

### Статистичні метрики

- Загальна кількість файлів
- Мінімальний / максимальний розмір
- Середнє арифметичне та медіана
- Перцентилі: P50, P75, P90, P95, P99
- Стандартне відхилення
- Розподіл по діапазонах (0-1KB, 1-10KB, 10-100KB, 100KB-1MB, 1-10MB, 10-100MB, 100MB+)

### Графіки

1. **Гістограма** (`plots/histogram.png`) — розподіл кількості файлів за розмірами з логарифмічною шкалою по осі X
2. **CDF** (`plots/cdf.png`) — кумулятивна функція розподілу з позначеними перцентилями
3. **Boxplot** (`plots/boxplot.png`) — діаграма розмахів на логарифмічній шкалі

## Висновки

*(Заповнюється після запуску аналізу)*

Переважна більшість файлів (**X%**) має розміри у діапазоні від **A** до **B**.

## Як запустити

```bash
# Встановити залежності
uv sync

# Зібрати дані (скан всієї файлової системи)
cd lab1-file-size-analysis
make collect

# Аналіз та візуалізація
make analyze
```

Або напряму:
```bash
uv run python dtr.py collect       # збір даних
uv run python dtr.py analyze       # аналіз
uv run python dtr.py watch [path]  # оригінальна функція спостереження
```
